### 理解したこと

- `for` 文は、回数が決まっている繰り返し処理に向いている
- `while` 文は、終了条件が来るまで繰り返す処理に向いている
- `sum += i` は `sum = sum + i` と同じ意味
- `%` は割った余りを求める演算子
- `i % 2 == 0` で偶数判定ができる
- `i % 3 == 0 && i % 5 == 0` で「3の倍数かつ5の倍数」を判定できる
- Pythonの `and` にあたるものは、Javaでは `&&`
- Pythonの `input()` に近いものとして、Javaでは `Scanner` を使う
- `scanner.nextInt()` で整数入力を受け取れる
- `scanner.close()` でScannerを閉じる
- `break` を使うとループを途中で終了できる
- `=` は代入、`==` は比較
- Javaのクラス名は `FizzBuzzUntil100` のように単語ごとに大文字にする
- `public class` の名前とファイル名は一致させる必要がある
