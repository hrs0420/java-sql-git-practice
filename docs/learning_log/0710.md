## 理解したこと

- Javaの配列は `int[] salesCounts = {10, 5, 3};` のように書く
- 配列の番号は0から始まる
- `salesCounts[0]` で1番目の値を取り出せる
- `salesCounts.length` で配列の要素数を取得できる
- `for (int i = 0; i < salesCounts.length; i++)` で配列全体を順番に処理できる
- 合計は `totalSales += salesCounts[i];` で計算できる
- 最大値は、仮の最大値を用意して、より大きい値が出たら更新する

## 詰まったこと・気づき

- 配列はPythonのリストに近いが、Javaでは型を指定する必要がある
- `i <= salesCounts.length` にすると範囲外アクセスになるため、`i < salesCounts.length` と書く必要がある
- 最大値を求める処理は、基本情報のアルゴリズム問題にもつながると感じた
